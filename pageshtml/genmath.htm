📍 Title: Polynomial Functions

📌 A. Definition

A polynomial function is a function of the form:
f(x) = anxn + an-1xn-1 + ... + a1x + a0
where coefficients ai are real numbers and n is a non-negative integer.

📌 B. Key Terms (short meanings)

  • Degree – highest exponent n of x.
  • Leading coefficient – an, coefficient of the highest degree term.
  • Constant term – a0, term without x.
  • Zero / Root – value of x where f(x)=0.

📌 C. Examples

  • 1) f(x) = 2x³ + 4x² − x + 7
    Degree: 3 — Leading coefficient: 2 — Type: Cubic
  • 2) g(x) = −5x² + 3x − 8
    Degree: 2 — Leading coefficient: −5 — Type: Quadratic
  • 3) h(x) = 7x⁵ − x⁴ + 2x² − 1
    Degree: 5 — Leading coefficient: 7 — Type: Quintic
  • Non-polynomial examples (NOT polynomial):
    • 1/x (variable in denominator)
    • √x (fractional exponent)
    • 5x⁻² + 3x + 1 (negative exponent)

📌 D. Domain

All real numbers (polynomials defined for every real x).

📌 E. End Behavior (based on degree and leading coefficient)

  • Even degree:
    • Positive leading coefficient → both ends up
    • Negative leading coefficient → both ends down
  • Odd degree:
    • Positive leading coefficient → left down, right up
    • Negative leading coefficient → left up, right down

📌 F. Finding Zeros (brief)

  • Factor the polynomial when possible (common factor, grouping, quadratic formula for degree 2).
  • Use Rational Root Theorem to test possible rational roots.
  • Use synthetic division or polynomial long division after finding a root.

📌 G. Worked Examples

  1. Example 1: f(x) = x² − 5x + 6
    Factor: (x − 2)(x − 3) → zeros: x = 2, 3. Degree 2, leading coeff = 1.
  2. Example 2: p(x) = (x − 1)(x + 2)(2x − 3)
    Expand if needed: degree 3, zeros: x = 1, −2, 3/2. Leading coeff = 2.
  3. Example 3: q(x) = 2x³ + x² − 5x − 2. Use rational root test: possible ±1, ±2, ±1/2.
    Test x=1 → 2+1−5−2 = −4 (no). Test x=2 → 16+4−10−2 = 8 (no). Test x=−1 → −2+1+5−2 = 2 (no). Test x=−2 → −16+4+10−2 = −4 (no). Test x=1/2 → 2(1/8)+1/4−5(1/2)−2 = 0.25+0.25−2.5−2 = −4 → no. If no rational roots, use numeric methods or graphing to approximate roots.
  4. Example 4 (End behavior): r(x) = −3x⁴ + 2x³ − x + 5 → Degree even (4), leading coeff negative (−3) → both ends down.

📌 H. Quick Practice

  1. Identify degree and leading coefficient of f(x) = 4x⁴ − x + 9.
  2. Factor and find zeros: x² + 7x + 10.
  3. Describe end behavior: f(x) = −x³ + 6x.

📌 I. Answer Key (short)

  • 1) Degree 4, leading coefficient 4.
  • 2) (x + 5)(x + 2) → zeros x = −5, −2.
  • 3) Odd degree, negative leading → left up, right down.

📍 Title: Rational Functions

📌 A. Definition

A rational function is a function of the form:
f(x) = P(x) / Q(x)
where P(x) and Q(x) are polynomial functions, and Q(x) ≠ 0.

📌 B. Key Terms (short meanings)

  • Numerator – the top polynomial P(x).
  • Denominator – the bottom polynomial Q(x).
  • Domain – all real x except values making Q(x) = 0.
  • Vertical Asymptote – vertical line x = a where Q(a) = 0 (and P(a) ≠ 0).
  • Horizontal Asymptote – horizontal line showing end behavior as x → ±∞.
  • Oblique/Slant Asymptote – occurs if degree of numerator is exactly 1 higher than denominator.

📌 C. Domain

To find the domain: Solve Q(x) = 0 and exclude these values from all real numbers.

📌 D. Types of Asymptotes

  • Vertical: at values excluded from domain.
  • Horizontal:
    • deg(P) < deg(Q) → y = 0
    • deg(P) = deg(Q) → y = leading coeff(P) / leading coeff(Q)
    • deg(P) > deg(Q) → no horizontal asymptote
  • Oblique/Slant: divide P(x) by Q(x) using long or synthetic division.

📌 E. Examples

  • Example 1: f(x) = (x + 1) / (x − 2)
    Domain: x ≠ 2
    Vertical asymptote: x = 2
    Horizontal asymptote: deg num = deg den = 1 → y = 1
  • Example 2: g(x) = (x² − 1) / (x − 3)
    Domain: x ≠ 3
    Deg num (2) > deg den (1) → slant asymptote found via division: y = x + 3 + 8/(x − 3)
  • Example 3: h(x) = 2 / (x² + 4)
    Domain: all real numbers (since x² + 4 ≠ 0 for real x)
    Horizontal asymptote: y = 0

📌 F. Graph Behavior

  • Near vertical asymptotes: graph shoots up to +∞ or down to −∞.
  • End behavior follows horizontal or slant asymptote.
  • May have holes if a factor cancels between P(x) and Q(x).

📌 G. Worked Example

  1. Example: f(x) = (x² − 9) / (x² − x − 6)
    Factor: (x − 3)(x + 3) / (x − 3)(x + 2)
    Hole at x = 3 (factor cancels)
    Vertical asymptote: x = −2
    Horizontal asymptote: deg num = deg den → y = 1

📌 H. Quick Practice

  1. Find domain of f(x) = (x + 4) / (x² − 16).
  2. Identify vertical and horizontal asymptotes of g(x) = (2x² + 1) / (x² − 9).
  3. State if h(x) = (x² + x + 1) / (x − 1) has a slant asymptote.

📌 I. Answer Key (short)

  • 1) x ≠ −4, 4.
  • 2) Vertical: x = 3, −3; Horizontal: y = 2.
  • 3) Yes, because degree numerator (2) is 1 greater than denominator (1).

📍 Title: Laws of Exponents

📌 A. Definitions

The laws of exponents are rules that simplify expressions involving powers of the same base. They allow easier manipulation of exponential terms.

📌 B. Main Laws & Formulas

  • 1. Product Rule: am × an = am+n
  • 2. Quotient Rule: am ÷ an = am−n
  • 3. Power of a Power: (am)n = am×n
  • 4. Power of a Product: (ab)m = am × bm
  • 5. Zero Exponent: a0 = 1 (where a ≠ 0)
  • 6. Negative Exponent: a−m = 1 / am

📌 C. Examples

  • Product Rule: 23 × 24 = 27 = 128
  • Quotient Rule: 56 ÷ 52 = 54 = 625
  • Power of a Power: (32)4 = 38 = 6,561
  • Power of a Product: (2 × 5)3 = 23 × 53 = 8 × 125 = 1,000
  • Zero Exponent: 70 = 1
  • Negative Exponent: 2−3 = 1 / 23 = 1/8

📌 D. Key Insights

  • Always apply laws only when the base is the same.
  • Negative exponents indicate reciprocals.
  • Zero exponent always equals 1 (except 00 which is undefined).

📍 Title: Radical Expressions

📌 A. Definitions

Radical expressions are expressions that contain a root symbol (√), such as square roots, cube roots, or higher-order roots. They can involve variables, constants, or both.

📌 B. Key Terms

  • Radical Sign (√): Indicates the root operation.
  • Radicand: The number or expression inside the radical sign.
  • Index: Indicates the degree of the root (e.g., index of 3 means cube root).
  • Principal Root: The non-negative root of a number.

📌 C. Laws of Radicals

  • Product Rule: √(ab) = √a × √b
  • Quotient Rule: √(a/b) = √a ÷ √b (b ≠ 0)
  • Power Rule: √[n]{am} = am/n
  • Simplifying: Factor perfect powers out of the radicand.

📌 D. Examples

  • Product Rule: √(4 × 9) = √36 = 6
  • Quotient Rule: √(25/4) = 5/2
  • Power Rule: √[3]{82} = 82/3 = 4
  • Simplifying: √50 = √(25 × 2) = 5√2

📌 E. Key Insights

  • Always simplify radicals by factoring out perfect powers.
  • Use the laws of radicals to combine or separate radical expressions.
  • Express radicals in rational exponent form for easier manipulation in algebra.

📍 Title: Future & Present Values of Simple and General Annuities

📌 A. Definitions

  • Simple Annuity – payments are made at regular intervals that coincide with the interest compounding periods.
  • General Annuity – payments occur at regular intervals that do not align with the compounding periods.

📌 B. Formulas

  • Simple Annuity:
    • Present Value: P = R × [ (1 – (1 + i)⁻ⁿ) / i ]
    • Future Value: F = R × [ ( (1 + i)ⁿ – 1 ) / i ]
    • Where:
    • R = periodic payment
    • i = interest rate per period
    • n = number of periods (n = t × m, where m = compounding periods per year)
  • General Annuity:
    • Present Value: P = R × [ (1 − (1 + i)⁻ⁿ) / ( (1 + i)ᵇ − 1 ) ]
    • Follows similar idea but adapts when payment and interest periods differ.

📌 C. Example – Simple Annuity

  • R = ₱2,000 quarterly for 5 years with 8% annual interest, compounded quarterly.
  • i = 0.08 / 4 = 0.02, n = 4 × 5 = 20.
  • Future Value: F = 2000 × [ ((1.02)²⁰ – 1) / 0.02 ] ≈ ₱55,474

📌 D. Example – General Annuity

  • Payments yearly; interest 5% compounded quarterly (payment ≠ compounding interval).
  • Convert interest: i = 0.05 / 4 = 0.0125; compute equivalent rate per year and apply general annuity formula.

📌 E. Key Insights

  • Present Value reflects the total worth now of future payments.
  • Future Value reflects total accumulation with interest over time.
  • Simple annuities are easier to compute when periods align.
  • General annuities require rate conversions when periods differ.
